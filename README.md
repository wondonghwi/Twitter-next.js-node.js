## Next.js + Node.js를 이용해 Twitter 만들기 Project

donghwi-reactbird.shop 도메인으로 이용

### [웹페이지 바로가기](http://donghwi-reactbird.shop/)

> Front-End
- ant-design
- next.js
- styledComponents
- Redux - Saga

> Back-End
- node.js
- express

> Aws EC2 배포 진행(중)
- front Ec2 서버 사용
- back Ec2 서버 사용
- S3 사용 (이미지 올리기)
- lambda 사용

> nginx를 이용해 https 주소를 사용할 수 있도록 서버쪽 작업 진행중 현재 코드작성보다 aws / nginx 작업 진행경험 후 제거
